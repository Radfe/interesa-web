# Interesa.sk - dalsia faza webu

Tento dokument urcuje, co ma ist dalej vo `web vlakne`, kym `admin vlakno` riesi produkty, odkazy a obrazky.

## 1. Co sa deje teraz

### Admin vlakno
Riesi:
- system produktov
- odkazy do obchodov
- obrazky produktov
- priradenie produktov ku clankom

### Web vlakno
Ma pokracovat v:
- obsahovej strategii
- SEO raste
- doplnani clankov
- rozsireni money pages
- internom prelinkovani

## 2. Najblizsie priority vo web vlakne

### P1 - dopisat a rozsirit najsilnejsie money clanky

1. `najlepsie-proteiny-2026`
- doplnit este silnejsie sekcie:
  - pre koho je WPC
  - pre koho je WPI
  - pre koho je vegan protein
  - casto kladene otazky

2. `protein-na-chudnutie`
- dotiahnut intent:
  - protein pri diete
  - co sledovat pri redukcii
  - ci je clear protein realne uzitocny

3. `kreatin-porovnanie`
- posilnit:
  - komu staci monohydrat
  - kedy ma HCl zmysel
  - kratke vyvratenie najcastejsich mytov

4. `doplnky-vyzivy`
- spravit z toho este silnejsi vstupny clanok pre bezneho navstevnika
- doplnit:
  - co ma zmysel ako zaklad
  - co je uz nadstavba
  - co vacsina ludi kupuje zbytocne

### P2 - dopisat supporting clanky s velkym SEO potencialom

1. `vitamin-d3-a-imunita`
- velmi dolezity supporting clanok pre imunitu

2. `horcik-ktory-je-najlepsi-a-preco`
- silny SEO clanok s dobrym affiliate presahom

3. `probiotika-a-travenie`
- dobry supporting clanok pre Symprove a probioticky cluster

4. `kedy-brat-kreatin-a-kolko`
- velmi dobry supporting clanok do kreatin klastra

5. `spalovace-tukov-realita`
- vyborny trust clanok pre chudnutie cluster

## 5. Co ma ist hned po dokonceni admin importu

Ked admin doda prve produkty, vo web vlakne sa ma hned skontrolovat:
- ci sedí poradie produktov v clanku
- ci sedí hlavna odporucana volba
- ci sedí wording pri CTA
- ci sedí comparison table
- ci sedí prepojenie na suvisiace clanky

## 6. Najblizsi prakticky postup

### Hned teraz
- cakat na prvy admin vystup pre produkty
- medzitym pripravovat dalsie clanky a rozsirene osnovy

### Potom
- po dodani prvych produktov dorovnat top money pages
- skontrolovat, ci affiliate flow na clankoch posobi prirodzene

### Dalsia faza
- dalsie nove clanky
- dalsie temy
- dalsie kampane len ak budu realne potrebne
