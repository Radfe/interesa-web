# Interesa.sk - odkial sa budu brat konkretne produkty

Tento dokument vysvetluje, ako sa dostaneme od kampani k realnym produktom na webe.

## 1. Kto rozhoduje

Konkretne produkty na verejnom webe urcuje web vlakno.

To znamena:
- web vlakno rozhodne, co sa ma zobrazit
- admin vlakno spravi system, aby sa to dalo vlozit, ulozit a zobrazit

## 2. Co viem dnes a co este neviem

### Dnes viem
- ktore kampane sa hodia na Interesa.sk
- ktore clanky su priorita
- ake typy produktov patria do danych clankov
- kolko produktov ma ist do boxov
- aka ma byt priorita obchodov

### Dnes este neviem kompletne
- presny zoznam produktov z kazdeho e-shopu po kusoch
- aktualnu dostupnost produktov
- presne nazvy, balenia a varianty v kazdom obchode
- ktore konkretne produkty budu mat najlepsie obrazky

Z toho plynie:
- ja nemam produkty slepo vymyslat
- potrebujem realny vstup z e-shopov alebo z admin importu

## 3. Z coho budem vychadzat pri vybere produktov

Konkretne produkty budem vyberat z tychto zdrojov:

1. Schvalene kampane a obchody
- GymBeam.sk
- Protein.sk
- IronAesthetics.sk
- Symprove.sk
- Imunoklub.sk
- Kloubus.sk

2. Produktove sloty pre top clanky
- tie uz mame pripravene v rollout dokumente

3. Realny zoznam produktov z obchodov
- ten musi prist:
  - z admin importu
  - alebo z exportu / feedu
  - alebo zo zoznamu URL produktov

## 4. Co je najlepsi sposob, ako mi dodat produkty

Najlepsie moznosti su tieto, od najlepsiej po najslabsiu:

### Moznost A - admin import kandidatskych produktov
Najlepsia moznost.

Admin vlakno pripravi jednoduchy import alebo manualne pridanie kandidatov pre:
- proteiny
- kreatin
- horcik
- D3 / C / zinok
- probiotika
- kolagen / klby
- pre-workout
- multivitamin

Potom ja vo web vlakne vyberiem, co sa ma realne zobrazit.

### Moznost B - export alebo feed
Ak vies ziskat feed alebo export produktov z danych kampani, je to velmi dobre.

Staci aj hruby export s tymito poliami:
- obchod
- nazov produktu
- URL produktu
- obrazok
- kategoria
- pripadne cena

### Moznost C - zoznam URL produktov
Ak nema admin este hotovy system, mozes mi dodat aj rucny shortlist URL.

Priklad:
- 10 az 20 produktov z GymBeam pre proteiny
- 6 az 10 produktov pre kreatin
- 5 produktov pre horcik

Aj to staci na prve kuratorstvo.

## 5. Co od teba potrebujem, aby som vedel vybrat konkretne produkty

Ak chces, aby som ti presne povedal, ktore konkretne produkty maju ist na web, potrebujem aspon jednu z tychto veci:

1. shortlist produktov z obchodov
2. export produktov
3. admin import kandidatov

Bez toho viem rozhodnut:
- typy produktov
- pocet slotov
- roly produktov
- prioritu obchodov

Ale neviem poctivo potvrdit:
- tento presny produkt ano
- tento presny produkt nie

## 6. Ako to spravime bez chaosu

Spravny postup je:

1. web vlakno urci:
- ktore clanky
- ake typy produktov
- kolko slotov
- z ktorych kampani

2. admin vlakno doda kandidatske produkty

3. web vlakno urobi finalny vyber:
- hlavna volba
- value volba
- alternativa
- porovnanie

## 7. Co urcite nechcem robit

Nechcem:
- slepo doplnat nahodne produkty bez realneho zoznamu
- rozhodovat len podla provizie
- naplnit web celym e-shopom

Chcem:
- kuratorstvo
- vyber len tych produktov, ktore realne sedia clanku
- rozumny launch-ready set

## 8. Co je teraz najlepsi dalsi krok

Najpraktickejsi dalsi krok je tento:

1. admin vlakno pripravi jednoduchy zoznam kandidatov pre prve hlavne temy
2. alebo mi ty dodas shortlist URL produktov z hlavnych obchodov
3. ja potom spravim presny vyber kus po kuse

To je najrychlejsia cesta k tomu, aby sme vedeli presne povedat:
- ktore vyrobky budu na stranke
- z ktorych obchodov
- v akom poradi
