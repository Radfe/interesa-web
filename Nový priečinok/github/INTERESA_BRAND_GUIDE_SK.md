# Interesa.sk Brand Guide

Tento brief je pripraveny pre rychlu pracu v Canva Pro.
Je zamerany na jemny refresh verejneho webu, nie na velky rebrand.

## Ciel

Vytvorit cistejsiu, zapamatatelnejsiu a doveryhodnejsiu vizualnu identitu pre Interesa.sk tak, aby:

- web posobil profesionalnejsie
- ostal redakcny a obsahovy
- favicon bola citatelna
- OG obrazky vyzerali konzistentne
- tema zdravia, vyzivy a porovnani posobila vecne, nie agresivne reklamne

## Odporucany smer

Najlepsi smer pre Interesa.sk je:

- zachovat motiv rastu, orientacie a porovnania
- zjednodusit symbol
- zjemnit farebnu paletu
- posilnit citatelnost v malych rozmeroch
- oddelit logo pre header od favicon verzie

Neodporuca sa:

- velky rebrand
- velmi hrave alebo startupove logo
- preplnene farebne symboly
- komplikovane gradienty vo favicon

## Odporucanie pre logo

Najlepsia volba:

- jemne vylepsit sucasne logo

Preco:

- aktualny smer je obsahovo spravny
- symbol uz naznacuje rast a porovnavanie
- problem nie je v myslienke, ale v prevedeni

## 3 navrhove smery

### Smer A - Cisty rast

Najviac odporucany smer.

Symbol:

- 3 alebo 4 jednoduche stlpce
- jeden jemny rastovy akcent
- bez drobnych detailov

Wordmark:

- Interesa.sk
- tmavy, cisty, bez experimentov

Pocit:

- doveryhodny
- redakcny
- moderny

### Smer B - Orientacia a vyber

Symbol:

- jednoduche stlpce + sipka alebo bod orientacie
- trochu viac editorial look

Pocit:

- viac obsahovy magazin
- menej datovy graf

### Smer C - Minimal ikonka

Symbol:

- uplne minimalisticka favicon verzia
- len 3 stlpce alebo 1 rastovy znak

Pocit:

- najlepsie funguje v malom
- vhodne ako sekundarna verzia, nie hlavne logo

## Finalne odporucanie

Pouzit:

- Smer A ako hlavne logo
- Smer C ako favicon a ikonovu verziu

## Farebna paleta

Hlavne farby:

- Forest Green: `#119A67`
- Deep Navy: `#10202D`
- Soft Mint: `#EAF6F0`

Doplnkove farby:

- Warm Coral: `#E55C3C`
- Leaf Green: `#20B16A`
- Slate Gray: `#5B6672`
- Light Border: `#DCE7E2`
- Background: `#F6FAF8`

Pouzitie:

- hlavny text: Deep Navy
- primarne CTA a pozitivny signal: Forest Green
- jemne pozadia: Soft Mint
- doplnkovy akcent: Warm Coral
- sekundarny text: Slate Gray

## Typografia

Ak budes robit grafiku v Canve:

- Nazov / hlavny nadpis: `Manrope ExtraBold` alebo `Nunito Sans ExtraBold`
- Doplnenie / podnadpis: `Manrope Medium` alebo `Nunito Sans SemiBold`

Neodporucam:

- velmi ozdobne fonty
- prilis tenke fonty
- technologicky alebo futuristicky styl

## Styl ikon

Odporucany styl:

- jednoduche line alebo flat ikony
- bez 3D efektov
- bez ilustracneho chaosu
- jemne zaoblene hrany

Ikony by mali posobit:

- prakticky
- cisto
- editorialne

## Co pripravit hned v Canve

Prva faza:

- logo horizontal
- logo icon
- favicon 32x32
- apple touch icon
- default OG obrazok

Druha faza:

- homepage OG
- OG template pre clanok
- OG template pre kategoriu
- 12 tematickych coverov

## Presne exporty

### Logo horizontal

- format: SVG + PNG
- odporucany canvas: 1200 x 300
- export PNG s transparentnym pozadim

### Logo icon

- format: SVG + PNG
- odporucany canvas: 512 x 512
- musi fungovat aj ako favicon zaklad

### Favicon

- PNG 32 x 32
- PNG 48 x 48
- idealne aj SVG verzia

### Apple touch icon

- PNG 180 x 180

### Default OG

- PNG
- 1600 x 900

### OG template pre clanok

- PNG
- 1600 x 900

Obsah:

- logo alebo symbol
- nazov webu
- hlavny nadpis clanku
- jemny podnadpis
- cisty editorial background

### OG template pre kategoriu

- PNG
- 1600 x 900

Obsah:

- symbol alebo male logo
- nazov kategorie
- kratky podnadpis
- jemny motiv podla temy

## Ako ma vyzerat OG styl

Vsetky OG obrazky maju mat:

- svetle jemne pozadie
- silny tmavy nadpis
- malu brand znacku
- jednoduchy shape alebo jemny motiv
- bez preplnenia

Homepage OG:

- hlavny claim webu
- vizual "vyber bez chaosu"

Clanok OG:

- nadpis clanku
- mala kategoriova znacka
- jednoduchy ilustracny alebo editorial motiv

Kategoria OG:

- nazov temy
- jedna jasna veta, co tam clovek najde

## 12 tem - vizualny system

Kazda tema ma mat:

- rovnaky layout
- vlastny akcent alebo motiv
- rovnaku typografiu
- rovnaky pomer stran

Priklad smerovania:

- Proteiny: mint + clean shaker / powder
- Vyziva: jemny food editorial
- Mineraly: ciste kapsuly / mikro ziviny
- Sila: treningovy detail / shaker / kreatin
- Klby a koza: collagen / clean neutral wellness
- Imunita: D3 / citrus / clean routine

## Co odovzdat admin vlaknu

Do systemu ma ist aspon toto minimum:

- logo-full
- logo-icon
- favicon-32
- apple-touch-icon
- og-default

Idealne hned potom:

- og-home
- og-article-template
- og-category-template
- theme-cover-01 az theme-cover-12

## Prakticky pracovny postup

1. V Canve sprav 2 verzie loga:
- hlavne horizontal
- zjednoduseny symbol

2. Zo symbolu odvod:
- favicon
- apple touch icon

3. Zo stylu loga odvod:
- default OG
- article OG template
- category OG template

4. Až potom rob 12 tematickych coverov.

## Kratky verdikt

Logo:

- nie nechat tak
- nie velky rebrand
- ano, jemny refresh

Favicon:

- urcite zjednodusit

OG obrazky:

- urcite prerobit na jednotny system

Najvyssia priorita:

- symbol
- favicon
- horizontal logo
- default OG
