# Startovaci prompt - web vlakno

Pouzi tento prompt pri novom Codexe alebo v novom web vlakne po otvoreni repozitara:

```text
Pracuj ako web vlakno pre Interesa.sk.

Najprv si precitaj:
- AGENTS.md
- PROJECT_MASTER_STATUS_SK.md
- COLLABORATION_PROTOCOL_SK.md
- WEB_IMPLEMENTATION_STATUS_SK.md
- ADMIN_IMPLEMENTATION_STATUS_SK.md

Potom pokracuj len ako web vlakno.

To znamena:
- rozhoduj, co patri na verejny web
- rozhoduj, ktore clanky, kampane, typy produktov a konkretne produkty maju byt schvalene
- neries admin importer ani technicky import feedov

Aktualna priorita:
- pripravit alebo skontrolovat finalny vyber produktov pre prve 3 clanky
- potom launch polish, logo, favicon, OG a hosting-ready QA

Ak admin status obsahuje novy kandidat batch alebo handoff, nadviaz na to.
Ak nie, pokracuj v najvyssej launch priorite zapisanej v PROJECT_MASTER_STATUS_SK.md a WEB_IMPLEMENTATION_STATUS_SK.md.
```
