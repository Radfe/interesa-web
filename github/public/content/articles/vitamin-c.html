Vitamin C - davky, zdroje a kedy ho doplnat

Zmysel vitaminu C v beznom zivote: kolko ho treba, kedy staci strava a kedy dava zmysel doplnok.

Rychly zaver

  • Vo vacsine beznych situacii nie je vitamin C zazracny doplnok, ale skor jednoduche doplnenie stravy.
  • Ak mas pestru stravu, casto ho vies pokryt aj bez doplnku.
  • Najcastejsia chyba je ist zbytocne do vysokych davok a cakat, ze vyriesi celu imunitu.

Co je dobre vediet na zaciatok

Vitamin C je jeden z najznamejsich doplnkov, ale prave preto sa pri nom casto ocakava viac, nez realne robi. Vo vacsine beznych situacii je to skor jednoduchy doplnok alebo doplnenie stravy, nie zazracne riesenie imunity a unavy.

Ako si zuzit vyber bez chaosu

  1. Najprv si povedz, ci vitamin C riesis pre slabu stravu alebo len ako jednoduchy doplnok do rutiny.
  2. Ak mas pestru stravu, casto netreba robit z vitaminu C velku vedu.
  3. Ak chces doplnok, drz sa skor zrozumitelnej a striedmej volby, nie co najvyssich cisel.

Kedy staci strava

Ak mas pravidelne ovocie, zeleninu a celkovo aspon trochu pestry jedalnicek, casto vies prijem vitaminu C pokryt prirodzene. Velky zmysel doplnok dava skor vtedy, ked vies, ze tvoj prijem zo stravy je slabsi alebo velmi jednostranny.

Kedy dava zmysel doplnok

  • ked mas dlhodobo nizky prijem ovocia a zeleniny
  • ked chces jednoduche doplnenie bez zbytocneho komplikovania
  • ked sa vitamin C objavuje ako sucast sirsej rutiny, nie ako jedina zachrana

Kolko dava zmysel

Bezne davky byvaju skor striedme. Ist dlhodobo vysoko len preto, ze je vitamin C popularny, casto nema velky prakticky prinos a niekomu moze zhorsovat travenie.

Kedy zbytocne nepriplacat

Ak len chces mat vitamin C ako jednoduchu sucast rutiny, nema zmysel kupovat najagresivnejsie alebo najviac marketingove verzie. Praktickejsie je mat zrozumitelny doplnok alebo si najprv upratat bezne zdroje v strave.

Najcastejsie chyby

  • ocakavat od vitaminu C, ze vyriesi celu imunitu
  • ignorovat zakladne veci ako spanok, stravu a rezim
  • ist zbytocne do vysokych davok bez dovodu

Casto kladene otazky

Treba brat vitamin C kazdy den? Nie vzdy. Zavisi od stravy a od toho, preco ho vlastne riesis.

Je viac vitaminu C vzdy lepsie? Nie. Vyssie davky automaticky neznamenaju vyssi uzitok.

Pomaha vitamin C pri imunite? Skor ako doplnok do sirsej rutiny, nie ako jedine riesenie.
