# Dognet Storage

Sem patria budúce exporty a feedy z Dognetu alebo od merchantov.

Odporúčaná štruktúra:
- `raw/` pre surové exporty
- `normalized/` pre interne spracované výstupy
- `logs/` pre import logy

Tieto feedy nie sú určené na verejné servovanie. Slúžia ako vstup pre budúci import workflow a normalizáciu produktových dát.
